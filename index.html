Pod Mapping Instructions

Pod mapping is a tool created by Mia Mingus to help people identify who they can turn to for various kinds of support.

How to Use This Tool

  • Choose a map type using the dropdown
  • Fill in your name and map details
  • Add pod members (solid-line circles)
  • Add potential members (dotted-line circles)
  • Add community resources or networks (larger circles)
  • Click on any element to select it, then use the Edit or Delete buttons
  • Save your map to return to it later (stored in your browser)
  • Export your map as a file to back up or share
